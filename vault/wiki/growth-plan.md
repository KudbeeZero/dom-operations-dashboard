# Organic Growth Plan — I Got A Dom

> Goal: the site brings in traffic + business on its own, and works as the
> instant-engagement landing when Dominick shows a business the QR code.
> Written Sprint 154 (2026-07-01). Owner actions vs HERMES actions marked.

## The funnel (already live)
QR / search / AI recommendation → igotadom.online → SMS 773-647-7598 (primary
CTA everywhere) → job. The QR code on the site itself opens a prefilled text.

## 1. Walk-in / QR play (owner — highest immediate ROI)
- [ ] Print the QR (assets QR → sms) on a card/sticker: "Messy menu? Old flyer?
      Text me a photo — clean version same day, $25." Hand it to the exact
      businesses whose documents you'd fix (restaurants = menus, salons = price
      lists, contractors = quotes/flyers).
- [ ] The pitch IS the demo: ask for their messiest doc, clean it same day,
      then offer the website ("your menu's clean — your site could look like this").
- [ ] Follow-up card for web leads: "I build sites you OWN — no agency retainer."

## 2. Google surface (owner — the #1 search lever, all free)
- [ ] Google Business Profile (service-area business, Chicago/IL) — categories:
      Website Designer + Graphic Designer + (closest fit) document services.
      This is what makes "web designer near me" show you.
- [ ] Search Console + Bing Webmaster: verify, submit sitemap.xml.
- [ ] After every happy job, text the client the GBP review link. Real reviews
      compound; we keep NO fake ones on-site, so Google reviews carry the proof.
- [ ] Free citations: Yelp, Nextdoor, Facebook page, Thumbtack/Bark (lead gen).

## 3. On-page SEO / AEO (HERMES — partly done Sprint 154)
- [x] JSON-LD: ProfessionalService + offers + FAQPage; knowsAbout now includes
      website design / graphic design / SEO / AI search optimization / mini apps.
- [x] FAQ "Do you build websites?" (FAQ rich-result + AI-assistant answerable).
- [x] Meta description covers documents + websites + AI optimization.
- [ ] NEXT: per-service landing anchors already exist (#web, #services,
      #pricing); if web-design leads grow, split /websites into its own page
      (own title/description) — single-page limits keyword coverage.
- [ ] NEXT: llms.txt + richer Service schema for the web tiers once prices are set.

## 4. AI-search moat (HERMES)
- The site itself is the proof-of-work for the AEO service: structured data,
  honest copy, fast load, chat assistant. "I did this for my own site" is the
  sales line. Screenshot/casestudy it once the chat bot goes live.

## 5. Content flywheel (owner + HERMES, later)
- Before/after gallery grows with every real job (with client OK) — that's the
  organic content engine; each example targets a query ("menu redesign",
  "resume cleanup"). No fake examples.

## Sequencing
Week 1: GBP + Search Console + print QR cards (owner). HERMES: nothing blocked.
Week 2+: reviews loop + citations. Revisit /websites page split when web
inquiries start arriving.
