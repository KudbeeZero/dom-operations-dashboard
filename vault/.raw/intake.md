# Raw Intake — I Got A Dom

> Drop messy notes, brand briefs, and unstructured client info here.
> HERMES reads this, structures it, and writes clean versions into /wiki/.

## Source

- Concept inspired by the @ridark_eth "Claude Code + Obsidian" architecture:
  Obsidian = persistent memory/wiki, Claude Code (HERMES) = the build agent.
- Applied to *I Got A Dom* — a same-day document cleanup service.

## Brand brief (raw)

- Name: I Got A Dom (never "I Got A Dom LLC")
- Tagline: Chaos to Clarity.
- What it does: messy notes, screenshots, menus, rough drafts go in →
  clean, professional, ready-to-send results come out. Same day.
- Tone: direct, confident, human, fast, premium. Not corporate, not scammy,
  not overpromising.
- Hard rule: this is NOT legal/tax/financial/professional advice. Just
  organizing facts, cleaning writing, building clear drafts.

## Notes / unstructured

- Pricing floated: $25 quick fix, $50 clean package (most popular), $75+ same-day buildout.
- Wants a dark "obsidian" aesthetic with teal + gold accents.
- Hero = three-stage "chaos to clarity" transformation animation.
